# Trading Dashboard

Persönliches Swing-Trading-Dashboard von Marten Scholz.

## Features
- COT-Reports (CFTC) inkl. COT-Index nach Williams (26 & 156 Wochen)
- Saisonalität (≥ 15 Jahre Historie)
- Leitzinsen & Zinsdifferenzen (FRED)
- Tägliche Aktualisierung via GitHub Actions

## Märkte
6E, 6B, 6J, 6A, 6C, GC, SI, CL, ES, NQ
